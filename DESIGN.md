# Zentra MY Design Context

## Brand Tokens

- Deep Navy: `#040043`
- Electric Blue: `#0021F3`
- Lavender Pearl: `#C1BFE3`
- Soft Lavender: `#F5F4FF`
- White: approved on navy and blue

## Typography

- Sora for display, headings, body.
- DM Mono for compact data, timestamps, and operational labels.
- Body copy should stay readable with generous line height.

## Design Direction

Business command board: dark, precise, high contrast, operational. The page should feel like a premium system already running behind the business, not a decorative tech poster.

## Component Guidance

- Use fewer transparent panels and stronger solid surfaces.
- Keep blue rare and decisive: CTAs, active journey states, key proof.
- Make proof sections feel like product evidence.
- Avoid decorative blur and excessive gradient text.
- Use clear contrast between copy and background.
